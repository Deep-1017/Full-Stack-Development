HTML - HyperText Markup Language 

Extention - .html, .htm

BoilerPlate Code = Necessary / Required Code 
    -> html:5   OR   shift + 1 


## Tag & Elements

# Tag -  <Deep>, <Hi>, <iant>, <button>
    -> Opening Tag:  <button>
    -> Closing Tag:  </button>


# Elements - Opening Tag + Content/Data + Closing Tag
<h1>Heading 1</h1>


## Attributes = Used to pass extra information
<h1 name="heading">


## Comments = Extra Information 




CSS = Cascading Style Sheets 

3 Ways to add CSS into document:
    1> Inline CSS
        <h1 style="background-color: red"> 

    2> Internal CSS
        <style>
            h2 {
                background-color: blue
            }
        </style>

    3> External CSS 
        - Make a new file "style.css"
        - Link css file to html

        h2 {
            background-color: red
        }
